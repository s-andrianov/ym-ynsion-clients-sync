# ym-ynsion-clients-sync
Прослойка, синхронизирующая клиентов-участников яндекс.музыки с клиентом-ведущим по Ynison

Работает так: 

Подключаемся к вебсокет-серверам (ynison) ведущего и участников.
Слушаем обновления состояния у ведущего. 
При получении обновления состояния переводим в формат управляющего сообщения и передаем участникам, меняя состояние их плееров.
